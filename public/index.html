AnimalBreedAPI is an API that provides the most common and popular breeds among the various pets in your household.

Documentation

GET

Gives a complete list of all the types of animals stored in the api. The list is returned in format JSON. Every object returned has an internal id and his name in English (EN).

/api/types

Request

Headers

  • API_KEY - The API Key that is needed to receive data from the application

Params

  • id - The id of the type of animal

Body

  • type - The the type of animal

Response
