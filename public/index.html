AnimalBreedAPI is an API that provides the most common and popular breeds among the various pets in your household.

Documentation

GET

Gives a complete list of all the breeds of animals stored in the api. The list is returned in format JSON. Every object returned has an internal id, breed information ant type. All information is in English (EN).

/api/breeds

Request

Params

  • id - The id of the type of animal (int)

Body

  • breed - The the breed of animal (string)

  • type - The the type of animal (string)

  • orderBy - The kind of value that is being ordered. Could be id, type or breed (string)

  • orderDirection - The direction of the order. ASC (ascending order) or DESC (descending order) (string)

  • limit - Limits the number of results

  • page - When given a limit, returns the page number x of results

Response


    {
        "response": [
            {
                "id": 4,
                "breed": "Gecko",
                "type": "reptile"
            },{
                "id": 990,
                "breed": "Pomsky",
                "type": "dog"
            }
        ],
        "res_size": 2,
        "res_time": "0ms"
    }
